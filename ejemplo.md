## ¿Qué es Docker?

Docker es una plataforma que permite ejecutar aplicaciones en contenedores.

## Contenedores vs Máquinas Virtuales

Una máquina virtual incluye su propio sistema operativo invitado.

## Imágenes y Contenedores

Una imagen es un plano de solo lectura. Un contenedor es una instancia.

## El Dockerfile

El Dockerfile es un archivo de texto con instrucciones para construir una imagen.

## Comandos esenciales

- docker build, docker run, docker ps, docker images

## Docker Compose

Docker Compose permite definir y ejecutar aplicaciones multi-contenedor.

## Buenas prácticas

Usá imágenes base pequeñas como Alpine.
